# Sales Data Analysis using Power BI & Power Query

## Project Overview
This project analyzes sales data to uncover trends, performance metrics, and actionable insights using Power BI and Power Query.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV

## Key Objectives
- Clean and transform raw sales data using Power Query
- Create interactive dashboards in Power BI
- Analyze sales trends, revenue, and product performance
- Provide business-ready insights for decision-making

## Dashboard Highlights
- Total Sales & Revenue KPIs
- Monthly & Yearly Sales Trends
- Top Products & Categories
- Region-wise Sales Performance

## Files Included
- `Sales_Data_Analysis.pbix` – Power BI dashboard
- `sales_data.csv` – Dataset
- `dashboard_overview.png` – Dashboard snapshot
- `sales_trends.png` – Sales trends visualization
